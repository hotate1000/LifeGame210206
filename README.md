### 遊び方
- ボックスをクリックし、任意のボックス色を黒または白に変更してください。
- 「進む」または「自動」ボタンをクリックし、黒ボックスの変化を確認し、遊びます。
  - 変化のルールは参考文献をご参照ください。
- 「自動」ボタンをクリックした際は、「停止」ボタンをクリックすることで、自動を解除出来ます。
- 「削除」ボタンをクリックした際は、画面がリセットされます。
